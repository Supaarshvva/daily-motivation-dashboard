# 🌞 Daily Motivation Dashboard

A responsive React + Tailwind CSS application that fetches motivational quotes and allows users to like, store, and search them.

---

## 🚀 Features

- 🔄 Fetch random quotes using API
- ❤️ Like / Unlike quotes
- 💾 Persist liked quotes using localStorage
- 🔍 Search through liked quotes
- ⏳ Loading state handling
- 📭 Empty state UI
- 📱 Responsive modern design

---

## 🧠 Concepts Used

- useState for UI & data management
- useEffect for API calls
- Conditional rendering
- Array `.map()` for rendering lists
- Unique keys for list items
- Controlled input for search
- Component-based architecture

---

## 🛠 Tech Stack

- React (Vite)
- Tailwind CSS v4
- Axios
- React Icons
- ESLint

---

## 🌐 API Used

https://dummyjson.com/quotes/random
